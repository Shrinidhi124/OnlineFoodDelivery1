# Online Food Delivery System

## Overview

The Online Food Delivery System is a Java-based web application developed using JSP, Servlets, JDBC, MySQL, HTML, and CSS. It provides a simple and user-friendly platform for browsing restaurants, exploring restaurant-specific menus, adding food items to a shopping cart, and placing orders.

The project demonstrates the implementation of Java Full Stack concepts, including dynamic web pages, database connectivity, session management, and CRUD operations.

---

## Features

- User Authentication
- Restaurant Listing
- Restaurant-Specific Menus
- Food Item Display with Images
- Add to Cart Functionality
- Shopping Cart Management
- Checkout Process
- Order Confirmation
- Responsive User Interface

## Installation and Setup

1. Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/OnlineFoodDelivery.git
```

2. Open the project in Eclipse IDE.

3. Configure Apache Tomcat 10.

4. Create the required MySQL database.

5. Update the database connection details in the project.

6. Run the project on Tomcat.

7. Open the application in your browser:

```text
http://localhost:8080/OnlineFoodDelivery/home.jsp
```

---

## Application Workflow

1. Launch the application.
2. Browse the list of restaurants.
3. Select a restaurant.
4. View the restaurant-specific menu.
5. Add food items to the shopping cart.
6. Review the cart.
7. Complete the checkout process.
8. Receive the order confirmation.
